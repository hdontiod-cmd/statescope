StateScope is a web application that allows users to explore the population data across the U.S. using the Data USA API.

Project Description

The purpose of this project is to simplify access to centers based on demographic information by transforming Raw API data into charts, tables, and visualizations.

Data Source

This application uses the datausa API which provides access to the United States Census and demographic dataset.

API Endpoint Used

https://api.datausa.io/tesseract/data.jsons
